## Regex para as mensagens

`(feat||fix|docs|style|refactor|test|chore)\(.*\)\:.*\`

## Prefixo

- feat
- fix
- docs
- style
- refactor
- test
- chore

seguido de `:`

## Mensagem

Letras em caixa baixa

## Exemplo

feat: organizacao de itens
